CSS Sizing Units : 
In CSS there are many sizing units:
    1. px : (1/96) inch fixed
    2. vw : Takes the viewport width(Screen width)
    3. vh : Takes the viewport height(Screen height)
    4. em : Inherits the font size from its parent element
    5. rem : Inherits the font size the root element(html)
    6. vmin : Takes the minimum between viewport width and viewport height
    7. vmax : Takes the maximum between viewport width and viewport height
    8. % : Takes the given percent of it's parents value
